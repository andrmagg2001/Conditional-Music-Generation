# Security Policy

## Supported Versions

We currently support security updates for the `main` branch. If you are using a specific version or branch, please update to the latest `main` commit to ensure you have the latest security patches.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them directly to the repository maintainers. You can do this by sending an email to the authors or reaching out via private communication channels if provided in the repository's main documentation. 

We will try to acknowledge your report within 48 hours, and provide an expected timeline for a fix. Please allow us time to release a patch before disclosing the vulnerability publicly.
